This is the About page for the project.

JavaScript has not been used in this project yet, as the main focus was on building and strengthening the backend functionality first. While I do understand the core concepts of JavaScript, I plan to dedicate more time to learning and applying it in future improvements to enhance interactivity across the website.

This was also my first time using Tailwind CSS, which is why the design may appear less refined or user-friendly in some areas. However, this was intentional as part of the learning process.

The goal of this project was to challenge myself by setting a limited timeframe and seeing how much functionality I could implement within a few weeks, prioritising backend development and working system features over perfect UI design.
