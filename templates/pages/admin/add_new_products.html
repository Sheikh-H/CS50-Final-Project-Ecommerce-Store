This is the Add Products page, where new products can be created and stored in the system.

When a product is added, the image upload is handled through Cloudinary, which stores the image in the cloud and returns a secure URL. This URL is then saved alongside the product data so it can be displayed throughout the website.

The application currently uses a local SQLite3 database file to manage and store all product and user data. Flask is used to connect the front-end of the website to the database, allowing data to be inserted, retrieved, and displayed dynamically.

A planned improvement is to migrate the database to an online solution such as Supabase, and to use SQLAlchemy as the ORM. This would improve scalability, make database management more flexible, and provide better integration for future development.
